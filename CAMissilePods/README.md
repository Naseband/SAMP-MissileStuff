# CAMissilePods by NaS

Adds the ability to attach a Missile/Bomb Pod to any Vehicle.

When a player is in the vehicle, they can shoot Missiles that launch from the Pod by using a predefined key.

Any type of Missiles from CAMissiles can be used. Homing functionality is still TBD.

Currently supported object models for Pods:

- 3790 (Small Missile)
- 3786 (Medium Missile)
- 1636 (Bomb)
